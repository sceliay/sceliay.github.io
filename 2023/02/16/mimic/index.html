MIMIC-III

  • 患者人口统计学信息与院内周转信息
    • PATIENTS: 患者信息
      • SUBJECT_ID, GENDER
      • DOB, DOD, DOD_HOSP, DOD_SSN
    • ADMISSIONS:患者入院信息
      • SUBJECT_ID, HADM_ID: patients id, admission id
      • ADMITTIME, DISCHTIME, DEATHTIME
      • ADMISSION_TYPE, ADMISSION_LOCATION
      • INSURANCE, LANGUAGE, RELIGION, MARITAL_STATUS, ETHNICITY
      • EDREGTIME, EDOUTTIME: emergency
      • DIAGNOSIS: preliminary, free text diagnosis
      • HOSPITAL_EXPIRE_FLAG: dead in hospital
    • ICUSTAYS: ICU入住信息
      • SUBJECT_ID, HADM_ID, ICUSTAY_ID
      • FIRST_CAREUNIT, LAST_CAREUNIT, FIRST_WARDID, LAST_WARDID
      • INTIME, OUTTIME, LOS
    • CALLOUT: 患者离开ICU信息
    • SERVICES: 医疗服务
    • TRANSFERS: 周转信息
  • 患者在监护室住院期间采集的信息
    • CAREGIVERS: 看护者信息
    • CHATEVENTS:电子图表，生命体征，护理相关的附加信息:呼吸机设置、实验室值、代码状态、精神状态等等。
      • SUBJECT_ID, HADM_ID, ICUSTAY_ID
      • ITEMID
      • CHARTTIME, STORETIME
      • CGID
      • VALUE, VALUENUM
    • DATETIMEEVENTS: 操作信息
    • INPUTEVENTS_CV: 入量信息
    • INPUTEVENTS_MV:
    • NOTEEVENTS: 治疗记录
    • OUTPUTEVENTS: 出量信息
    • PROCEDUREEVENTS_MV: 操作或手术信息
  • 医院记录系统采集的信息
    • CPTEVENTS: 现行程序术语(current procedural terminology, CPT)
      • SUBJECT_ID, HADM_ID
      • COSTCENTER: 支付中心，ICU支付或Resp对应于机械或无创通气，由呼吸治疗师收费
      • CHARTDATE
      • CPT_CD, CPT_NUMBER, CPT_SUFFIX
      • TICKET_ID_SEQ
      • SECTIONHEADER, SUBSECTIONHEADER: CPT代码分类
    • DIAGNOSES_ICD: ICD diagnoses
      • SUBJECT_ID, HADM_ID
      • SEQ_NUM, ICD9_CODE
    • DRGCODES: 诊断类别组
    • LABEVENTS: 化验项目
    • MICROBIOLOGYEVENT: 微生物病原体检测
    • PRESCRIPTIONS: 用药记录
    • PROCEDURES_ICD: 手术记录
  • 字典信息
    • D_CPT.csv: 操作记录
    • D_ICD_DIAGNOSES.csv：诊断
    • D_ICD_PROCEDURES.csv: 手术代码
    • D_ITEMS.csv：记录项目
    • D_LABITEMS: 化验项目
